## 1. BRAND POSITIONING

### Core Positioning Statement

**For** solo operators, indie hackers, and lean agencies who need autonomous AI execution without SaaS lock-in or cloud dependency,
**DA Agency / Ghost Legion** is a **local-first multi-agent operating system** that runs entirely on your machine — not someone else's cloud.

Unlike generic AI SaaS tools that charge per-seat and trap your data, Ghost Legion is a **sovereign agent fleet** you own, command, and deploy. It is the only multi-agent ecosystem that treats your desktop as the command center, not a browser tab.

### Positioning Dimensions

| Axis | Where We Sit | What We Reject |
|------|-------------|----------------|
| **Deployment** | Local-first, desktop-native (Electron) | Cloud-only, browser-locked SaaS |
| **Philosophy** | Sovereign compute — you own the stack | Platform dependency, API lock-in |
| **Interaction Model** | Directives, not dialogs — command-to-execution | Chat-first, manual-heavy workflows |
| **Aesthetic** | Cyberpunk operational theater — coral on black | Clean SaaS minimalism, corporate blue |
| **Scale** | One operator + agent fleet = full agency | Team-size pricing, per-seat models |

### The "Dual Domain" Insight

Decio operates across two territories:
- **Business Projects** (local files, local execution, private IP)
- **Web Projects** (deployed assets, client deliverables, public presence)

Ghost Legion is the bridge. It keeps sensitive work local (Business Projects) while generating deployable output (Web Projects). This is not a bug — it is the value proposition. The brand must communicate: *your command center stays dark; only the output goes live.*

## 2. VOICE & TONE

### Voice Attributes

| Attribute | Definition | In Practice |
|-----------|-----------|-------------|
| **Operational** | Every word implies action and status | "Agent spawned" not "Task created" |
| **Classified** | Information is earned, not given freely | "Signal classified" — context over explanation |
| **Surgical** | No fluff. Precision over persuasion | "Route to grid" not "Please assign to team" |
| **Alive** | The system has agency, not just functionality | "Fleet online" not "System is running" |
| **Direct** | Imperative mood, active voice | "Issue directive" not "You can now send a message" |

### Tone Matrix by Context

| Context | Tone | Example |
|---------|------|---------|
| **Onboarding** | Guided but terse | "Initialize your operatives. The grid awaits." |
| **Success states** | Confirmed, not celebrated | "Directive executed. 3 tasks routed." |
| **Errors** | Diagnostic, not apologetic | "Bridge degraded. Check HERMES_PATH." |
| **Marketing** | Provocative, insider | "Your competitors use chatbots. You command a legion." |
| **Documentation** | Reference manual, not tutorial | "Endpoint: /api/hermes/spawn. Method: POST." |

### Forbidden Language

- "AI-powered" — every tool claims this. We are agent-native.
- "Seamless" — implies invisibility. Our interface is intentionally present.
- "Unlock potential" — motivational fluff. We route tasks.
- "Democratizing" — political language. We arm operators.
- "Simply" / "Just" — condescending. If it were simple, we would not exist.

### Typography

| Role | Font | Weight | Tracking | Case |
|------|------|--------|----------|------|
| **Headlines / UI Labels** | Inter | 900 (Black) | 0.18em | UPPERCASE |
| **Data / Monospace** | JetBrains Mono | 400-700 | 0.12em | UPPERCASE for labels |
| **Body** | Inter | 400-500 | 0.02em | Sentence case |
| **Micro-labels** | JetBrains Mono | 400 | 0.2em | UPPERCASE |

### Visual Principles

1. **Scanlines are not decoration — they are evidence.** The CRT overlay signals this is a terminal, not a webpage. It communicates: *you are inside the machine.*

2. **Hexagons are the core glyph.** The G01 logo uses a hex clip-path. Hexagons = hive intelligence, modular architecture, tactical mapping.

3. **Opacity is information.** Online agents glow at 100%, idle at 50%, offline at 30%. The visual system encodes state without requiring labels.

4. **Borders are structural, not decorative.** Every panel has a 1px `rgba(255,255,255,0.08)` border. This creates the "glass-in-darkness" effect — surfaces float in void space.

5. **Animation is functional.** The pulse on the HERMES ONLINE dot, the blink on the briefing cursor, the slide on the command palette — these are status indicators, not embellishments.

## 5. KEY MESSAGING PILLARS

### Pillar 1: Sovereign Compute

**Core Message:** Your agents run on your machine. Your data never leaves. You are not a tenant — you are the landlord.

**Proof Points:**
- Electron app loads from `file://` — no web server
- Bridge binds to `127.0.0.1:8767` — localhost only
- No API keys to external services required
- Whisper voice dictation uses local model (optional)

**Taglines:**
- "The cloud is someone else's computer. The Legion is yours."
- "No subscriptions. No telemetry. No surrender."
- "Your fleet. Your machine. Your rules."

### Pillar 2: Operational Velocity

**Core Message:** From directive to execution in seconds. Not chat threads. Not ticket queues. Direct command-to-action.

**Proof Points:**
- Create task → Hermes kanban → Agent claim → Execution (one pipeline)
- Cron jobs auto-spawn agents on schedule
- Command palette (⌘K) for instant navigation
- Task search (⌘F) for immediate retrieval

**Taglines:**
- "Issue the directive. The fleet handles the rest."
- "Less chat. More action."
- "Your operatives don't sleep. Neither does the grid."

### Pillar 3: Unified Command

**Core Message:** One deck for agents, tasks, cron, comms, and intelligence. No tab switching. No context loss.

**Proof Points:**
- 10 modules in one window: Command, Network, War Room, Operations, Comms, Factory, Briefing, etc.
- Live data across all modules from single Hermes bridge
- Legion roster visible in sidebar at all times
- Bridge diagnostics one click away

**Taglines:**
- "One operator. One deck. Infinite operatives."
- "Everything visible. Everything routed. Everything controlled."
- "The only dashboard you need is the one that never leaves your machine."

### Pillar 4: Extensible Architecture

**Core Message:** Built to be hacked. Open bridge. Clear types. Add your own modules. Ghost Legion is a chassis, not a cage.

**Proof Points:**
- Bridge is 87 lines of Python — readable in one sitting
- Every endpoint maps 1:1 to a Hermes CLI command
- React + TypeScript + Zustand — standard stack
- Demo modules show how to wire new data sources

**Taglines:**
- "Built to be extended. Designed to be owned."
- "The bridge is open. Cross it."
- "Not a product. A platform you pilot."
